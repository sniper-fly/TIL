ECMAscriptとは？			標準化されたjavascript. 現在はjs = ecmascript
DOMとは？					HTMLやXML文書を扱いやすくするための手法。document object model
jsonとは？					javascript object notation.構造化データの記述言語。
jsonの用途は？				web上でのデータのやり取り。XMLよりデータファイルが小さく、速い
フィードとは？				ウェブページの更新情報を配信するためのファイル。登録しておけば、ページにアクセスしなくても最新情報が得られる。
マイクロフォーマットとは？	HTMLで記述されたウェブページの中に意味を埋め込むための書式。
エンコード、デコードとは？	エンコードは情報圧縮、デコードは解凍、伸張すること。
